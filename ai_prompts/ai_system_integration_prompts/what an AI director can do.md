What AI Director SHOULD control (safe + doable)
Enemy count → +1 / -1
Enemy composition → (more ranged / more swarm etc.)
Spawn timing → faster / slower waves
Spawn pattern → spread / ambush / tighter clusters
Reinforcement chance → extra enemies mid-fight
Hazard density → slightly ↑ / ↓ lava/slow %
Reward drop chance → more help when struggling
Safe room strength → better upgrades / more heal
⚔️ Boss / Mini-boss tuning (keep LIMITED)
Attack frequency (cooldowns ↓ / ↑)
Telegraph duration (longer = easier)
Movement speed (small ± change)

❌ Avoid:

changing attack types
changing core mechanics
🧠 Player experience tuning
Pressure level (how often enemies engage)
Recovery windows (time between waves)
Difficulty pacing across rooms
⚠️ What NOT to do (important)
❌ Change RNG / seed logic
❌ Break room structure
❌ Modify core combat system
❌ Add new mechanics dynamically
🎯 Golden rule

👉 AI Director = small adjustments, not redesign
